Pengertian OOP

Object Oriented Programming (OOP) adalah paradigma pemrograman yang menggunakan konsep objek dan class untuk menyusun struktur program.

Konsep Dasar OOP

  • Class
    • Merupakan blueprint atau cetakan untuk membuat objek.
  • Object
    • Merupakan hasil dari pembuatan class.
  • Encapsulation
    • Menggabungkan data dan method dalam satu objek.
  • Inheritance
    • Pewarisan sifat dari class lainnya.
  • Polymorphism
    • Satu method dapat mempunyai banyak bentuk.
