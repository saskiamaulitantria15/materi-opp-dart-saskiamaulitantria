Pengertian OOP

Object Oriented Programming (OOP) adalah paradigma pemrograman yang menggunakan konsep objek dan class untuk menyusun struktur program.

Konsep Dasar OOP

  • Class
    • Adalah blueprint atau cetakan untuk membuat objek.
  • Object
    • Adalah hasil dari pembuatan class.
  • Encapsulation
    • Adalah konsep menyembunyikan data.
  • Inheritance
    • Adalah pewarisan sifat dari class induk.
  • Polymorphism
    • Adalah kemampuan method memiliki banyak bentuk.

Kesimpulan

Secara keseluruhan, dengan menggunakan konsep Object Oriented Programming (OOP), program dapat dibuat lebih terstruktur, mudah dipahami, dan mudah dikembangkan dalam jangka panjang.
